Politique de confidentialité

Application : Saphir Beauté

Dernière mise à jour : 2026

1. Introduction

Saphir Beauté respecte votre vie privée. Cette politique explique comment nous collectons, utilisons et protégeons vos données lorsque vous utilisez notre application.

2. Données collectées

Nous pouvons collecter les données suivantes :

  • Informations personnelles (nom, numéro de téléphone, email si fourni)
  • Informations de compte utilisateur (Firebase Authentication)
  • Données de rendez-vous (date, heure, service, notes)
  • Contenu utilisateur (messages, photos envoyées)
  • Données techniques (identifiants Firebase, logs d’utilisation, données de l’appareil)

3. Utilisation des données

  • Gestion des rendez-vous
  • Communication entre le client et le gérant
  • Stockage et affichage des réservations
  • Amélioration de l’application
  • Sécurité et prévention des fraudes

4. Stockage des données

Les données sont stockées de manière sécurisée via Firebase, incluant : Firebase Authentication, Cloud Firestore et Firebase Storage.

5. Partage des données

Les données sont uniquement accessibles dans le cadre de l’application :

  • Entre le client et le gérant pour la gestion des rendez-vous
  • Via les services Firebase utilisés pour le fonctionnement de l’application
Nous ne vendons aucune donnée personnelle.

6. Paiements et données financières

Les paiements ne sont pas effectués dans l’application. Ils sont réalisés directement au salon. L’application permet uniquement l’enregistrement et le suivi interne des revenus.

7. Permissions

L’application peut demander :

  • 📷 Caméra : prise de photos
  • 🖼️ Galerie : sélection d’images
  • 🌐 Internet : synchronisation des données
  • 📍 Localisation (uniquement pour afficher le salon sur une carte, sans suivi de position)

8. Sécurité des données

Nous utilisons Firebase Security Rules et des mesures de sécurité standard pour protéger les données des utilisateurs.

9. Droits des utilisateurs

Les utilisateurs peuvent demander l’accès, la modification ou la suppression de leurs données à tout moment.

10. Conservation des données

Les données sont conservées tant que nécessaire pour assurer le fonctionnement de l’application et la gestion des rendez-vous.
